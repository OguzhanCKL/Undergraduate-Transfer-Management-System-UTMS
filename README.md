# Undergraduate Transfer Management System (UTMS)

## 📝 Overview
[cite_start]UTMS is a centralized web platform developed for **Izmir Institute of Technology (IZTECH)** to digitize and automate the undergraduate transfer process[cite: 2714, 3105]. [cite_start]It replaces sequential manual workflows with an integrated digital system for students, administrative staff (ÖIDB), and academic commissions (YGK)[cite: 2431, 3198, 3199].

---

## 🚀 Key Features

* [cite_start]**Automated Eligibility:** Automatically validates GPA, credits, and semester requirements based on IZTECH and YÖK regulations[cite: 2624, 2625, 3109, 3144].

* [cite_start]**Document Management:** Secure upload and multi-stage verification for transcripts and official documents[cite: 2617, 2618, 3107, 3108].

* [cite_start]**Academic Ranking:** Automated GPA conversion and department-based applicant ranking[cite: 2631, 2633, 3110].

* [cite_start]**Role-Based Access Control (RBAC):** Specialized dashboards for Students, ÖİDB staff, YGK members, and Faculty Board members[cite: 2653, 2922].

## 🛠️ Tech Stack

* [cite_start]**Frontend:** React + Vite [cite: 2886][cite_start], Zustand[cite: 2893].

* [cite_start]**Backend:** Node.js[cite: 2902].

* [cite_start]**Database:** PostgreSQL [cite: 2913] [cite_start]with Drizzle ORM[cite: 2907].

* [cite_start]**Authentication:** JWT (JSON Web Token)[cite: 2919].
